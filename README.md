## 技术栈
vue2 + vuex + vue-router + webpack + ES6/7 + less + vant + axios

## 说明
此项目为客户端，另有laravel编写的服务端，因为只为学习前端我就不上传了，直接运行此项目会有数据找不到报错。

可以将此项目给作为一个前端脚手架使用
